# 문서 갱신 규칙

문서 작성/갱신 시 `.manuals/process/doc-update.md`를 읽고 따를 것.

## 핵심 (목차만)
- README.md: 프로젝트 최상위 문서. 구조, 사용법, 설치 반영.
- src/*/README.md: 모듈별 문서. 역할, 인터페이스, 의존성.
- .manuals/: 내부 지식. 규칙 변경 시 반영.
- 상태 스냅샷: 02_context.md 하단에 세션 상태 기록.
- 상세 규칙은 `.manuals/process/doc-update.md` 참고
