# 실험 로그 템플릿

새 실험 시 이 폴더를 복사하여 `YYYYMMDD-실험명/`으로 생성.

## 폴더 구조

```
YYYYMMDD-실험명/
├── config.yaml      # 실험 파라미터 (하이퍼파라미터, 데이터 경로 등)
├── env.yaml         # 환경 스냅샷 (python, torch, cuda, gpu, os)
├── run.sh           # 재현 커맨드 (복붙하면 동일 실험 재현)
└── result.md        # 실험 기록
```

## result.md 필수 섹션

```markdown
## 실험 대상
- 파일: (수정/실험한 파일 경로)
- 목적: (이 실험으로 확인하려던 것)
- 배경: (왜 이 실험이 필요한지)

## 실험 내용
- 변경 사항 요약

## 결과
- 수치/로그/스크린샷

## 관찰 / 교훈
- 다음에 참고할 점
```
