# /check — CCTV 품질 검증

현재 체크리스트 항목의 완료 상태를 검증합니다. reviewer 에이전트를 호출합니다.

## 검증 대상
`$ARGUMENTS`가 있으면 해당 파일/모듈을 검증. 없으면 가장 최근 완료된 체크리스트 항목 대상.

## 검증 항목

### 필수 (실패 시 블록)
1. 테스트 코드 존재 여부 (`tests/` 미러링 확인)
2. 테스트 통과 여부 (`pytest` 실행)
3. import 절대경로 준수 (`from src.` 패턴)
4. 하드코딩 설정값 없음

### 권장 (경고만)
1. 컨벤셔널 커밋 형식
2. 미러링 폴더 생성 여부
3. `.plans/03_checklist.md` 업데이트 여부
4. 실행 로깅 포함 여부

## 완료 후
- 통과: "다음 항목 진행 가능" 안내
- 실패: 구체적 수정 사항 제시, 수정 후 재검증
