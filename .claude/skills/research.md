# 리서치 규칙

리서치 수행 시 `.manuals/process/research.md`를 읽고 따를 것.

## 핵심 (목차만)
- 소스 우선순위: 공식문서 > arXiv > GitHub issues > Reddit
- 출력: 요약 + URL + 신뢰도(공식/커뮤니티/미확인) + 날짜
- 못 찾으면 못 찾았다고 보고. 추측 금지.
- 상세 포맷은 `.manuals/process/research.md` 참고
